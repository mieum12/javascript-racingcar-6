# 게임의 흐름

1. **자동차 이름 입력**

-`경주할 자동차 이름을 입력하세요.(이름은 쉼표(,) 기준으로 구분)`출력

- 자동차 이름 입력
- 예외 시 에러 출력

2. **시도할 횟수 입력**

- `시도할 횟수는 몇 회인가요?`출력
- 시도할 횟수 입력
- 예외 시 에러 출력

3. **게임 실행**

- 자동차마다 랜덤 값 생성 후 4 이상이면 전진( `-`로 표시)

4. **게임 종료**

- `최종 우승자 : ${우승자}`출력
- 우승자가 여러명일 경우 ,를 이용하여 구분

# 🏎️ 기능 요구 사항

## [domain]

### 1. Car/Cars

비즈니스 로직을 실행할 도메인 객체들

1. 각 차량을 움직임
2. CarDto/CarsDto 생성

### 2. MoveDecider

자동차 움직임 여부 판단

### 3. Dto

CarDto, CarsDto, WinnerDto, WinnersDto

## [view]

### 1. InputView

사용자 입력 처리 담당

### 2. OutputView

게임 메세지 및 결과 출력 담당

### 3. InputValidator

입력값 검증

### 4. InputConverter

입력값 변환

## [controller]

### 1. GameController

게임의 전체적인 흐름 담당
