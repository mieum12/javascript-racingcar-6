# 🏎️ 기능 요구 사항

## [domain]

### 1. Car

자동차 이름 저장
이동 가능 여부 판단
자동차가 이동한다면 총 몇번 이동했는지 세기

### 2. Racing

시도할 횟수 저장
각 자동차마다 레이스 한번씩 돌리기, 결과 저장
최종 레이스 종료 후 결과값 반환
최대 이동거리 계산
우승자 반환

### 2. OutputView

게임 메세지 및 결과 출력 담당

### 3. InputValidator

입력값 검증

### 4. InputConverter

입력값 변환

### 1. GameController
